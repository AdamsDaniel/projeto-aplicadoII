# StatusCheck - Frontend

Este é o frontend do projeto StatusCheck, uma aplicação React para gerenciar e visualizar o status de treinamentos de colaboradores.

## Visão Geral

O StatusCheck é uma plataforma que permite o acompanhamento de treinamentos, oferecendo uma interface intuitiva para colaboradores e gestores. O frontend é construído com React e utiliza componentes para criar uma experiência de usuário interativa e responsiva.

### Funcionalidades

- **Login de Usuário**: Autenticação segura para acesso à plataforma.
- **Dashboard Interativo**: Gráficos e visualizações de dados sobre o status dos treinamentos.
- **Gerenciamento de Colaboradores**: Cadastro e consulta de informações de colaboradores.
- **Consulta de Treinamentos**: Visualização dos treinamentos disponíveis e do progresso dos colaboradores.

## Começando

Estas instruções fornecerão uma cópia do projeto em funcionamento na sua máquina local para fins de desenvolvimento e teste.

### Pré-requisitos

O que você precisa para instalar o software:

- [Node.js](https://nodejs.org/) (versão 14.x ou superior)
- [npm](https://www.npmjs.com/) (geralmente vem com o Node.js)

### Instalação

Siga os passos abaixo para configurar o ambiente de desenvolvimento:

1. **Navegue até o diretório do frontend:**
   ```sh
   cd StatusCheckProjetoIntegrado
   ```
2. **Instale as dependências:**
   ```sh
   npm install
   ```
## Scripts Disponíveis

No diretório do projeto, você pode executar:

### `npm start`

Executa a aplicação no modo de desenvolvimento.
Abra [http://localhost:3000](http://localhost:3000) para visualizá-la no navegador.

A página será recarregada se você fizer edições.
Você também verá quaisquer erros de lint no console.
Você pode aprender mais na [documentação do Create React App](https://facebook.github.io/create-react-app/docs/getting-started).

Para aprender React, confira a [documentação do React](https://reactjs.org/).
